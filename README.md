# SD CareQueue

Community Clinic Appointment and Queue Management System

## Features
- Appointment booking
- Queue management
- Real-time tracking
- Clinic search (SA dataset)
- Analytics dashboard

## Tech Stack
- HTML, CSS, JavaScript
- Firebase Authentication (Google Sign-In)
- Firestore (Database)
- Jest (Testing)
- GitHub Actions (CI/CD)
